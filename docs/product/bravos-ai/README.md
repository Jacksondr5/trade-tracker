# Bravos AI Docs

This directory contains evergreen product documentation for Bravos AI features in Trade Tracker.

Use these docs when working on the Bravos-grounded assistant, knowledge base, or related future AI capabilities.

## Docs

- `system-map.md` - broad system map, workstreams, and `V1` / `V1.5` / `V2` scope boundaries
- `knowledge-base.md` - purpose, boundaries, and design direction for the Bravos-grounded AI assistant
- `knowledge-representation.md` - how Bravos source material should be segmented, represented, cited, and inspected
- `future-scope.md` - future-facing Bravos AI work beyond the core `V1` assistant loop
