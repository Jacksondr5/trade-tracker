# Product Docs

This directory contains evergreen product documentation for Trade Tracker.

Unlike the dated documents in `docs/plans/`, these docs are meant to stay current and describe the product as a whole rather than a single feature effort.

## Current docs

- `product-vision.md` - mission, north star, intended outcomes, and product boundaries
- `product-principles.md` - durable product rules for feature, workflow, and UX decisions
- `information-architecture.md` - core objects, relationships, overlays, and intentional flexibility in the model
- `ux-principles.md` - interaction and workflow rules for clarity, context, review, and efficient use
